## 🎯 Мета проєкту

Дослідження українського ринку відеоспостереження як живого датасету.
Збір даних → очищення → EDA → висновки.

## 📈 Ключові висновки

- **Hikvision** домінує — 40.2% ринку за кількістю моделей
- **Ajax** — найдорожчий сегмент, середня ціна **6,556 грн**
- **Imou** — найбюджетніший, середня ціна **1,713 грн**
- Медіана ринку: **3,306 грн**
- 81% камер є в наявності

## 🛠 Технології

- Python 3.14
- Pandas — обробка та аналіз даних
- Matplotlib — візуалізація
- BeautifulSoup — збір даних
